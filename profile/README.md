## Getting Started

1. Install the Autel-Explorer app on your mobile device.  
2. Connect your Autel drone via Wi-Fi or USB.  
3. Open the app and set up your drone for first flight (calibration/configuration).  
4. Configure camera and flight settings, plan mission if needed.  
5. Fly, record footage, monitor telemetry and manage updates.
